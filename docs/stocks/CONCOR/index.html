System Behaviour Commentary

For CONCOR, the system generated 76 trades over the full sample, averaging about 3.3 trades per year. The typical holding period is around 37.5 bars. Overall, this behaves like a very selective, long-term system, with very conservative risk, balanced between robustness and return.
