System Behaviour Commentary

For TRENT, the system generated 62 trades over the full sample, averaging about 2.7 trades per year. The typical holding period is around 38.9 bars. Overall, this behaves like a very selective, long-term system, with very conservative risk, balanced between robustness and return.
