System Behaviour Commentary

For CEATLTD, the system generated 72 trades over the full sample, averaging about 3.1 trades per year. The typical holding period is around 30.8 bars. Overall, this behaves like a very selective, long-term system, with very conservative risk, balanced between robustness and return.
