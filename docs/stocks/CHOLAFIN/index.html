System Behaviour Commentary

For CHOLAFIN, the system generated 50 trades over the full sample, averaging about 2.2 trades per year. The typical holding period is around 40.5 bars. Overall, this behaves like a very selective, long-term system, with very conservative risk, balanced between robustness and return.
