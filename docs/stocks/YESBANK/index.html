System Behaviour Commentary

For YESBANK, the system generated 61 trades over the full sample, averaging about 3.0 trades per year. The typical holding period is around 29.4 bars. Overall, this behaves like a very selective, long-term system, with very conservative risk, balanced between robustness and return.
