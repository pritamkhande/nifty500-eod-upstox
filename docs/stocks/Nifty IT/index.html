System Behaviour Commentary

For Nifty IT, the system generated 57 trades over the full sample, averaging about 2.2 trades per year. The typical holding period is around 26.1 bars. Overall, this behaves like a very selective, long-term system, with very conservative risk, balanced between robustness and return.
