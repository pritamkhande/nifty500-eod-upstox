System Behaviour Commentary

For BANDHANBNK, the system generated 26 trades over the full sample, averaging about 3.4 trades per year. The typical holding period is around 33.5 bars. Overall, this behaves like a very selective, long-term system, with very conservative risk, balanced between robustness and return.
