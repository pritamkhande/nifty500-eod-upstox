System Behaviour Commentary

For Nifty, the system generated 61 trades over the full sample, averaging about 2.3 trades per year. The typical holding period is around 27.2 bars. Overall, this behaves like a very selective, long-term system, with very conservative risk, balanced between robustness and return.
