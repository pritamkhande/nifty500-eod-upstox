System Behaviour Commentary

For THERMAX, the system generated 56 trades over the full sample, averaging about 2.4 trades per year. The typical holding period is around 38.2 bars. Overall, this behaves like a very selective, long-term system, with very conservative risk, balanced between robustness and return.
