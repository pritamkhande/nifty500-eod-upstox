System Behaviour Commentary

For NESTLEIND, the system generated 51 trades over the full sample, averaging about 3.2 trades per year. The typical holding period is around 35.0 bars. Overall, this behaves like a very selective, long-term system, with very conservative risk, balanced between robustness and return.
