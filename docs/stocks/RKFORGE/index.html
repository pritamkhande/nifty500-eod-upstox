System Behaviour Commentary

For RKFORGE, the system generated 67 trades over the full sample, averaging about 3.1 trades per year. The typical holding period is around 38.1 bars. Overall, this behaves like a very selective, long-term system, with very conservative risk, balanced between robustness and return.
