System Behaviour Commentary

For SBIN, the system generated 75 trades over the full sample, averaging about 3.3 trades per year. The typical holding period is around 31.6 bars. Overall, this behaves like a very selective, long-term system, with very conservative risk, balanced between robustness and return.
