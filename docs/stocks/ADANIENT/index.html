System Behaviour Commentary

For ADANIENT, the system generated 64 trades over the full sample, averaging about 2.8 trades per year. The typical holding period is around 29.4 bars. Overall, this behaves like a very selective, long-term system, with very conservative risk, tilted towards maximising return.
